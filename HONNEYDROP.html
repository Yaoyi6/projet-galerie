HONNEY DROP

Bienvenue sur cette page dédiée au projet Honney Drop.

DESCRIPTION

Il s'agit d'un jeu réalisé dans le cadre du projet tutoré S2.
Ce jeu est inspiré du jeu télévisé The Million Pound Drop. Le but du jeu est de répondre à toutes les questions en gardant le plus d'argent possible à la fin (l'argent est le miel dans cette version). Chaque questions aura quatre réponses possibles sur lesquels il faudra miser sur ce qui vous semble être la meilleure réponse. L'argent misé sur une mauvaise réponse est perdu. Il y a huit questions au total et le jeu s'arrête â l'issu de ces huit questions où â l'épuisement du miel possédé.
