DESCRIPTION

Il s'agit d'un jeu réalisé dans le cadre du projet tutoré S2.
Le but de ce jeu pédagogique est de deviner un mot à partir de la langue sélectionnée. Si vous réussissez, le jeu vous affichera sa traduction dans trois langues différentes.
